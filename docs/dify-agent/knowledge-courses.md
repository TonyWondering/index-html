# 码上成长 — 课程资源汇总

## 平台简介
"码上成长"是一个编程教育课程聚合平台，汇集前端、后端、AI、深度学习等多个方向的免费B站课程资源。目标是为学习者提供清晰的课程导航、收藏管理和学习进度追踪。

## 学习路径推荐

### 前端开发路线
1. HTML + CSS 基础 → 2. JavaScript 基础 → 3. Vue3 框架 → 4. TypeScript → 5. Element Plus 组件库 → 6. 项目实战

### 后端开发路线
1. Java 基础 → 2. MySQL 数据库 → 3. Spring Boot → 4. MyBatis → 5. Redis → 6. 项目实战

### AI/深度学习路线
1. Python 基础 → 2. NumPy/Pandas/Matplotlib → 3. PyTorch 入门 → 4. Transformer/CNN/LSTM → 5. 大模型API → 6. Agent开发
