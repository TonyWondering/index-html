# Dify 智能体搭建指南

## 第一步：注册 Dify 账号

1. 打开 https://dify.ai
2. 点击右上角 "Get Started"
3. 用邮箱注册（推荐 Gmail / Outlook / QQ邮箱）
4. 验证邮箱后登录，进入工作台

## 第二步：创建聊天助手应用

1. 在工作台点击 **"创建应用"**
2. 选择 **"聊天助手"**
3. 填写：
   - 应用名称：`码上成长助手`
   - 应用描述：`编程学习答疑与课程导航助手`
4. 点击"创建"

## 第三步：配置系统提示词

1. 进入应用后，左侧找到 **"编排"** 区域
2. 在 **"提示词"** 框中，粘贴 `SYSTEM_PROMPT.md` 文件的全部内容
3. 模型选择：推荐 `GPT-4o-mini`（性价比高）或 `GPT-4o`（效果更好）
4. 上下文轮数：设为 `10`

## 第四步：创建知识库并上传文档

1. 在左侧导航栏点击 **"知识库"**
2. 点击 **"创建知识库"**
3. 名称：`码上成长课程库`
4. 上传文件：选择 `knowledge-courses.md`
5. 分段设置：选择"自动分段"，分段长度 500，重叠 50
6. 索引方式：选择"高质量索引"
7. 等待索引完成（通常 1-2 分钟）

## 第五步：将知识库关联到应用

1. 回到应用编排页面
2. 在 **"上下文"** 区域点击 **"添加"**
3. 选择刚创建的 `码上成长课程库` 知识库
4. 检索模式：选择 **"混合检索"**
   - TopK：设为 `5`
   - Score 阈值：设为 `0.5`

## 第六步：配置开场白和建议问题

在应用设置中：

**开场白：**
```
你好！我是小码，码上成长平台的学习助手 😊

我可以帮你：
• 推荐适合你的编程课程
• 解答编程学习中的疑问
• 规划学习路径
• 找到你需要的软件工具

有什么想学的，尽管问我！
```

**建议问题（快捷入口）：**
1. 零基础应该先学什么？
2. 推荐一些前端入门课程
3. Python 和 Java 该选哪个？
4. 如何搭建后端开发环境？

## 第七步：获取嵌入代码

1. 在应用页面右上角点击 **"发布"**
2. 发布后，点击 **"访问"** 确认应用正常运行
3. 回到应用页面，左侧点击 **"嵌入"**
4. 选择 **"嵌入网站"** → **"JavaScript"**
5. 复制生成的代码，格式类似：

```html
<script>
  window.difyChatbotConfig = {
    token: '你的TOKEN',
    baseUrl: 'https://api.dify.ai',
  }
</script>
<script src="https://udify.app/embed.min.js" id="你的TOKEN" defer></script>
```

6. 将 token 和 baseUrl 记下来，告诉我，我帮你替换到网站代码中

## 第八步：测试

1. 在 Dify 应用页面右侧的预览窗口测试
2. 尝试问以下问题验证：
   - "我想学前端，应该从哪门课开始？"
   - "Python 数据分析用什么库？"
   - "推荐一些后端开发课程"
   - "在哪里下载 VS Code？"
